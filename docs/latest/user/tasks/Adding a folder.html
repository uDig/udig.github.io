Adding a folder¶

There are several ways to add folders to the Legend View.

Adding a folder using the toolbar button¶

  1. Click on the Add folder button on the view’s toolbar
  2. Your new folder will be added on top of the list

Adding a folder using the context menu¶

  1. Right-click in the Legend View
  2. Select Add folder from the context menu
  3. Your new folder will be added on top of the list
