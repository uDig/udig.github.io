Use a processing region as the AOI¶

We can use a processing region to define the current AOI.

  1. Open the AOI view
  2. Select the Processing Region option form the AOI combo box
  3. In the Palette open the Other tools option
  4. Select the Processing Region Box Selection tool
  5. On the map left click and drag to define your processing region and the current AOI.
