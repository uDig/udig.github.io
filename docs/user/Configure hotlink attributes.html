Configure hotlink attributes¶

We can mark attributes to store file or web hotlinks.

  1. Use Catalog view to select the resource
  2. Open the Resource Properties from the Data menu.
  3. Choose the Document property page
  4. Enable hotlink support using the provided checkbox
  5. List the attributes you wish to use to store hotlink information
    • Use the Add... to add an attribute as hotlink
    • Use the Edit... to update an existing hotlink definition
    • Use the Remove to remove an attribute as hotlink

A sample dataset is provided:

  • Try out australia.shp
  • File, Web and Action hotlinks are already defined
