Submit Error Log¶

The error log in uDig can be submitted to the uDig development team in 3 easy clicks.

  1. Select Help ‣ Submit Log... from the menu bar to open the Send Log dialog
  2. Type in some notes describing the nature of the problem and the steps required to reproduce it.
  3. You may also include contact information.
  4. Press the Proceed button.

The uDig bug tracker (JIRA) is available at: http://jira.codehaus.org/browse/UDIG

If the issue you have uncovered is not present here, please consider adding it. You may also “watch” an issue to learn when it has been resolved.
